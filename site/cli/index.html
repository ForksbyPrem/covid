    CLI (New)

    Get all data

    John Hopkins source (default)

    ./cli.py covid

    or

    ./cli.py covid -s john_hopkins

    Worldometers source

    ./cli.py covid -s worldometers

    List Countries

    This comes in handy when you need to know the available names of countries when using ./cli.py -s 'source' -c 'country_name', eg. "The Republic of Moldova" or just "Moldova" So use this when you need to know the country exact name that you can use.

    ./cli.py covid -s worldometers --list-countries

    Get Status By Country Name

    ./cli.py covid -s worldometers -c sweden

    Get Total Active cases

    ./cli.py covid -s worldometers -o active

    Get Total Confirmed cases

    ./cli.py covid -s worldometers -o confirmed

    Get Total Recovered cases

    ./cli.py covid -s worldometers -o recovered

    Get Total Deaths

    ./cli.py covid -s worldometers -o deaths
